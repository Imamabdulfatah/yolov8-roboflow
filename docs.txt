Berikut adalah langkah-langkah untuk membuat model YOLOv8 dengan menggunakan dataset gambar cabai dari Roboflow:

1. Siapkan Dataset Gambar Cabai di Roboflow
Kunjungi Roboflow.
Buat akun dan buat new project dengan nama sesuai kebutuhan (misalnya "Cabai Detection").
Upload dataset gambar cabai ke proyek tersebut, dan pastikan anotasi menggunakan format yang didukung oleh YOLO (misalnya COCO atau Pascal VOC).
Setelah selesai upload, ekspor dataset dalam format YOLO v5 PyTorch TXT (YOLOv8 juga kompatibel dengan format ini). Pilih ukuran dataset (misalnya 640px) dan salin API link untuk digunakan nanti.
2. Instalasi Python dan YOLOv8
Jika belum memiliki Python terinstall, unduh dan instal dari python.org.

Buat virtual environment (opsional, tapi disarankan):

bash
Copy code
python -m venv yolov8-env
source yolov8-env/bin/activate  # Untuk Linux/Mac
yolov8-env\Scripts\activate.bat  # Untuk Windows
Instal ultralytics library untuk YOLOv8:

bash
Copy code
pip install ultralytics
3. Download Dataset dari Roboflow
Setelah mengekspor dataset dari Roboflow, Anda akan menerima API URL untuk dataset Anda. Salin URL tersebut.

Untuk mendownload dataset:

bash
Copy code
pip install roboflow
Gunakan skrip Python berikut untuk mendownload dataset Anda:

python
Copy code
from roboflow import Roboflow

rf = Roboflow(api_key="YOUR_API_KEY")
project = rf.workspace("your-workspace").project("cabai-detection")
dataset = project.version(1).download("yolov8")
Setelah skrip dijalankan, dataset YOLO akan terunduh dan siap untuk digunakan.

4. Latih Model YOLOv8
Siapkan konfigurasi pelatihan YOLOv8, sebagai contoh buat file train.yaml dengan struktur seperti ini:

yaml
Copy code
train: ./dataset/train/images  # Lokasi data train
val: ./dataset/valid/images   # Lokasi data validasi

nc: 1  # Jumlah kelas, untuk cabai 1
names: ['cabai']  # Nama kelas
Mulai proses pelatihan dengan YOLOv8:

bash
Copy code
yolo task=detect mode=train data=path/to/train.yaml epochs=100 imgsz=640
epochs=100 berarti model akan dilatih selama 100 epoch.
imgsz=640 berarti gambar akan disesuaikan menjadi resolusi 640x640 piksel selama pelatihan.
5. Validasi dan Evaluasi Model
Setelah pelatihan selesai, Anda dapat mengevaluasi model dengan menggunakan data validasi:
bash
Copy code
yolo task=detect mode=val model=runs/detect/train/weights/best.pt data=path/to/train.yaml
6. Prediksi dengan Model Terlatih
Setelah model dilatih, Anda dapat menggunakannya untuk mendeteksi gambar cabai:

bash
Copy code
yolo task=detect mode=predict model=runs/detect/train/weights/best.pt source=path/to/test/images
Output hasil deteksi akan disimpan di folder runs/detect/predict.

7. Simpan Model Terlatih (Optional)
Jika ingin menggunakan model di tempat lain, Anda bisa menyimpan model tersebut:

Model terlatih akan disimpan di folder runs/detect/train/weights/ dengan nama best.pt.
Itu adalah langkah-langkah utama untuk membuat model YOLOv8 menggunakan data cabai dari Roboflow.




Berikut adalah langkah pertama secara lebih detail untuk membuat model YOLOv8 dengan dataset gambar cabai menggunakan Roboflow:

1. Siapkan Dataset Gambar Cabai di Roboflow
A. Buat Akun di Roboflow
Buka Roboflow.
Klik tombol Sign Up di kanan atas layar.
Isi informasi yang diperlukan untuk membuat akun baru (email, password) atau daftar menggunakan akun Google atau GitHub.
Setelah berhasil membuat akun, Anda akan masuk ke dashboard Roboflow.
B. Buat Proyek di Roboflow
Di dashboard Roboflow, klik tombol Create New Project.

Pada layar pop-up yang muncul, lakukan langkah berikut:

Project Name: Beri nama proyek, misalnya "Cabai Detection".
Project Type: Pilih jenis anotasi yang ingin Anda gunakan. Untuk kasus YOLO, pilih "Object Detection" (Deteksi Objek).
Licensing: Pilih lisensi yang relevan (misalnya, "Public" jika dataset Anda terbuka untuk umum atau "Private" jika dataset bersifat pribadi).
Klik tombol Create Project untuk melanjutkan.

C. Upload Dataset Gambar Cabai
Setelah proyek berhasil dibuat, Anda akan diarahkan ke halaman proyek. Di sini, klik Upload Images untuk mulai mengunggah gambar cabai.

Pilih semua gambar cabai yang ingin Anda gunakan dalam pelatihan YOLOv8. Pastikan dataset Anda mencakup berbagai sudut, pencahayaan, dan jenis cabai (misalnya, cabai rawit, cabai merah, dsb.).

Contoh:

Jika Anda memiliki 100 gambar cabai, pilih semua gambar tersebut untuk diunggah.
Setelah gambar diunggah, Roboflow akan meminta Anda untuk mengunggah anotasi untuk gambar tersebut (label bounding box). Jika Anda belum memiliki anotasi, Roboflow menyediakan fitur untuk melakukan anotasi secara manual.

D. Anotasi Gambar (Jika Belum Ada Anotasi)
Jika Anda belum memiliki anotasi, klik Annotate Images untuk memulai proses anotasi.

Pilih salah satu gambar cabai, lalu gunakan alat anotasi (kotak seleksi) untuk menandai objek cabai dalam gambar tersebut.

Setelah objek cabai terdeteksi, tambahkan label "Cabai" atau label lain yang sesuai.

Ulangi proses ini untuk semua gambar yang ada di dataset.

Contoh:

Jika Anda memiliki 100 gambar cabai, Anda perlu membuat anotasi bounding box untuk setiap cabai di dalam gambar tersebut.
E. Membagi Dataset (Opsional)
Setelah gambar dan anotasi selesai diunggah, Roboflow akan menawarkan pilihan untuk membagi dataset menjadi training, validation, dan test set.

Training Set: Biasanya 70-80% dari total gambar.
Validation Set: Biasanya 10-15% dari total gambar.
Test Set: Biasanya 10-15% dari total gambar.
Contoh:

Jika Anda memiliki 100 gambar, bagikan sebagai berikut: 70 gambar untuk training, 15 gambar untuk validation, dan 15 gambar untuk testing.
F. Ekspor Dataset dalam Format YOLO
Setelah dataset siap, klik Generate di pojok kanan atas untuk mengekspor dataset.

Pilih YOLOv5 PyTorch TXT sebagai format output (YOLOv8 kompatibel dengan format YOLOv5).

Pilih resolusi gambar yang diinginkan (misalnya 640px) untuk memperkecil ukuran dataset jika diperlukan.

Setelah itu, Roboflow akan menghasilkan link untuk mendownload dataset Anda.

Salin API Key atau link untuk digunakan nanti dalam pelatihan YOLOv8.

Contoh API Link:

bash
Copy code
https://app.roboflow.com/ds/your-dataset-link
G. Unduh Dataset
Anda dapat mendownload dataset secara langsung menggunakan link API tersebut.
Untuk Python, Anda juga bisa mendownload dataset melalui skrip Python (seperti dijelaskan di langkah selanjutnya).
Ini adalah langkah-langkah rinci yang perlu dilakukan pada langkah pertama untuk menyiapkan dataset cabai di Roboflow. Jika Anda mengikuti langkah-langkah ini dengan teliti, dataset akan siap untuk digunakan dalam pelatihan model YOLOv8.
